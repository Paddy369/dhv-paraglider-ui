# DHV Paraglider UI

A React-based web dashboard to explore, filter, and visualize paragliders registered in the official DHV database. The tool also allows the user to sort paragliders based on the optimized pilot start weight.

## Installation Guide

**Prerequisites:** Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

**1. Clone the repository**
```bash
git clone https://github.com/Paddy369/dhv-paraglider-ui.git
cd dhv-paraglider-ui
```

**2. Install dependencies**
```bash
npm install
```

**3. Start the development server**
```bash
npm run dev
```

## Data Source
The paraglider specifications visualized in this project were gathered from the official [DHV (Deutscher Hängegleiterverband) Device Database](https://service.dhv.de/db1/technicsearchpage.php).
